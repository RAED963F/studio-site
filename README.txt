ПАПКА ДЛЯ ФОТО ПОРТФОЛИО
=========================

Сюда нужно положить ваши фотографии для бегущей ленты на сайте
(раздел "Портфолио", между "Чем мы занимаемся" и пакетами услуг).

ЧТО НУЖНО СДЕЛАТЬ:
1. Назовите файлы цифрами по порядку: 1.jpg, 2.jpg, 3.jpg … 25.jpg
2. Положите их прямо в эту папку "photos" — рядом с index.html
   (структура должна получиться такой):

   index.html
   photos/
     1.jpg
     2.jpg
     ...
     25.jpg

РЕКОМЕНДАЦИИ ПО ФОТО (не обязательно, но для лучшего результата):
- Формат: JPG
- Ориентация: вертикальная (портретная), соотношение сторон примерно 4:5
- Минимальный размер: 800×1000 px (лучше — 1200×1500 px и больше)
- Желательно, чтобы все фото были похожи по цветовой гамме — лента
  выглядит наряднее, когда кадры сочетаются друг с другом

ЕСЛИ ФОТО ПОКА МЕНЬШЕ 25:
Ничего страшного — на месте отсутствующего файла сайт аккуратно покажет
плашку с номером кадра вместо "битой" картинки. Как только вы добавите
файл с нужным номером, он сам появится на сайте — ничего в коде менять
не нужно.

ЕСЛИ ХОТИТЕ ДРУГОЕ КОЛИЧЕСТВО ФОТО ИЛИ ДРУГОЕ НАЗВАНИЕ ПАПКИ:
Откройте index.html, найдите в самом низу файла (в <script>) строки:

  const GALLERY_FOLDER = "photos/";
  const TOTAL_IMAGES = 25;

Поменяйте название папки или число — и лента подстроится автоматически.
